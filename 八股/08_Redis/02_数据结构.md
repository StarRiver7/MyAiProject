# 数据结构（核心面试知识）

---

## 一、String（字符串）

### 1.1 特点

- 最基本的数据类型
- 可以存储字符串、数字、二进制数据
- 最大长度512MB

### 1.2 常用命令

```bash
# 设置值
set key value
setex key 60 value  # 设置过期时间
setnx key value     # 不存在才设置

# 获取值
get key

# 自增自减
incr key      # 自增1
incrby key 5  # 自增5
decr key      # 自减1
decrby key 5  # 自减5

# 字符串操作
append key "append"  # 追加
strlen key           # 长度
substr key 0 3       # 截取

# 批量操作
mset key1 value1 key2 value2
mget key1 key2
```

### 1.3 使用场景

- 缓存热点数据
- 计数器
- 分布式锁（setnx）
- 限流（incr + expire）

---

## 二、Hash（哈希表）

### 2.1 特点

- 键值对的集合
- 适合存储对象
- 节省内存

### 2.2 常用命令

```bash
# 设置字段
hset key field value
hmset key field1 value1 field2 value2

# 获取字段
hget key field
hmget key field1 field2
hgetall key

# 删除字段
hdel key field1 field2

# 字段操作
hlen key          # 字段数量
hexists key field # 字段是否存在
hkeys key         # 所有字段
hvals key         # 所有值

# 自增
hincrby key field 5
```

### 2.3 使用场景

- 存储用户信息
- 商品信息
- 配置信息
- 会话管理

---

## 三、List（列表）

### 3.1 特点

- 有序链表
- 可以在两端操作
- 支持阻塞操作

### 3.2 常用命令

```bash
# 左侧操作
lpush key value1 value2  # 左侧插入
lpop key                 # 左侧弹出
lrange key 0 -1          # 查看所有元素

# 右侧操作
rpush key value1 value2  # 右侧插入
rpop key                 # 右侧弹出

# 长度操作
llen key

# 阻塞操作
blpop key 10  # 阻塞10秒
brpop key 10  # 阻塞10秒

# 其他操作
lindex key 0  # 获取指定索引元素
lset key 0 value  # 设置指定索引元素
lrem key 2 value  # 删除2个value
ltrim key 0 4     # 保留指定范围
```

### 3.3 使用场景

- 消息队列
- 任务队列
- 最近访问列表
- 粉丝列表

---

## 四、Set（集合）

### 4.1 特点

- 无序集合
- 元素唯一
- 支持集合操作

### 4.2 常用命令

```bash
# 添加元素
sadd key member1 member2

# 查看元素
smembers key
sismember key member  # 元素是否存在

# 删除元素
srem key member1 member2
spop key  # 随机弹出

# 集合操作
sinter key1 key2       # 交集
sunion key1 key2       # 并集
sdiff key1 key2        # 差集

# 其他操作
scard key        # 元素数量
srandmember key  # 随机元素
```

### 4.3 使用场景

- 去重
- 标签系统
- 好友关系
- 共同好友

---

## 五、Sorted Set（有序集合）

### 5.1 特点

- 有序集合
- 元素唯一
- 每个元素有分数
- 按分数排序

### 5.2 常用命令

```bash
# 添加元素
zadd key score1 member1 score2 member2

# 查看元素
zrange key 0 -1 withscores  # 按分数升序
zrevrange key 0 -1 withscores  # 按分数降序

# 分数操作
zincrby key 5 member  # 增加分数
zscore key member      # 获取分数

# 范围操作
zrangebyscore key 1 10 withscores  # 按分数范围
zcount key 1 10                     # 分数范围内元素数量

# 删除操作
zrem key member1 member2
zremrangebyrank key 0 4  # 按排名删除
zremrangebyscore key 1 10  # 按分数删除

# 排名操作
zrank key member      # 升序排名
zrevrank key member   # 降序排名
```

### 5.3 使用场景

- 排行榜
- 延迟队列
- 优先级队列
- 时间序列数据

---

## 六、Bitmap（位图）

### 6.1 特点

- 位操作
- 节省空间
- 适合布尔值存储

### 6.2 常用命令

```bash
# 设置位
setbit key offset value  # value为0或1

# 获取位
getbit key offset

# 位操作
bitcount key  # 统计1的数量
bitop AND destkey key1 key2  # 位与
bitop OR destkey key1 key2   # 位或
bitop XOR destkey key1 key2  # 位异或
bitop NOT destkey key        # 位非
```

### 6.3 使用场景

- 签到统计
- 用户在线状态
- 布隆过滤器
- 权限控制

---

## 七、HyperLogLog

### 7.1 特点

- 基数统计
- 占用空间小
- 误差率低（约0.81%）

### 7.2 常用命令

```bash
# 添加元素
pfadd key element1 element2

# 统计基数
pfcount key

# 合并
pfmerge destkey key1 key2
```

### 7.3 使用场景

- 日活统计
- UV统计
- 独立访客统计
- 去重计数

---

## 八、Geospatial（地理位置）

### 8.1 特点

- 地理位置存储
- 距离计算
- 范围查询

### 8.2 常用命令

```bash
# 添加位置
geoadd key longitude latitude member

# 距离计算
geedist key member1 member2 [unit]  # unit: m, km, mi, ft

# 范围查询
geradius key longitude latitude radius unit [WITHCOORD] [WITHDIST] [WITHHASH]
geradiusbymember key member radius unit

# 位置信息
gepos key member1 member2
```

### 8.3 使用场景

- 附近的人
- 位置搜索
- 距离计算
- 地理围栏

## 九、数据结构对比

| 类型 | 有序性 | 唯一性 | 特点 | 适用场景 |
|------|--------|--------|------|----------|
| String | 无 | 键唯一 | 基础类型 | 缓存、计数器 |
| Hash | 无 | 字段唯一 | 对象存储 | 用户信息 |
| List | 有序 | 不唯一 | 链表 | 消息队列 |
| Set | 无序 | 唯一 | 集合操作 | 去重、标签 |
| Sorted Set | 有序 | 唯一 | 分数排序 | 排行榜 |
| Bitmap | 无 | 位唯一 | 位操作 | 签到、状态 |
| HyperLogLog | 无 | 无 | 基数统计 | 日活、UV |
| Geospatial | 无 | 唯一 | 地理位置 | 附近的人 |
